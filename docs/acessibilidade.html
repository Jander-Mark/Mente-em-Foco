Declaração de Acessibilidade

O projeto "Mente em Foco" está comprometido em garantir a acessibilidade digital para pessoas com deficiência. Estamos continuamente melhorando a experiência do usuário para todos e aplicando os padrões de acessibilidade relevantes.

As medidas tomadas para tornar este site mais acessível incluem:

  • HTML Semântico: Utilizamos tags HTML semânticas para estruturar o conteúdo, o que ajuda tecnologias assistivas a entender e navegar pelo site.
  • Texto Alternativo: Todas as imagens de conteúdo possuem um texto alternativo (alt text) para que usuários de leitores de tela possam compreender a informação visual.
  • Navegação pelo Teclado: O site é navegável utilizando apenas o teclado.
  • Contraste de Cores: Procuramos utilizar uma combinação de cores que ofereça um bom contraste entre o texto e o fundo.

Este é um projeto acadêmico e estamos aprendendo. Se você encontrar alguma barreira de acessibilidade, por favor, nos informe para que possamos considerar melhorias futuras.
